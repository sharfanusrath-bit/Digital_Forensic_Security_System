# Digital Forensic Secure Storage System

This is a Digital Forensic Security System designed to provide secure storage
and AI-based authentication for digital evidence.

Features:
- User Registration and Login
- Role-based access (Admin/User)
- Secure data storage
- AI-based Face Authentication (OpenCV)
- Activity logging

Technologies Used:
- Python
- OpenCV
- SQLite
- Cryptography
- Flask

Project Status: Under Development
